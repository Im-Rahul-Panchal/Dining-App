### Overview

This app allows users to:

- Book restaurant tables according to the number of people.

- View booking history user-wise to track past reservations.

- Manage bookings efficiently with a simple and intuitive interface.

### Features

- User authentication and profile management.

- Table availability checking in real-time.

- Booking confirmation with details of date, time, and number of guests.

- History of previous bookings for each user.

- Responsive design for mobile.

### Tech Stack:
- Frontend & Backend: React Native with Firebase
- Database & Authentication: Firebase Firestore and Firebase Auth

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start  OR  npm start
   ```
